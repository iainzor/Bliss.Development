The Docs module provides a way of compiling all available documentation into a single place. All registered modules can provide their own documentation simply by creating a /docs directory and filling it with HTML documents.

How it works

Each HTML file within the /docs directory acts as its own page; this also includes all sub-directories. This directory itself can be treated as its own set of web pages, including any type of asset.

In order to access the module's documentation, navigate to /docs/[module-name]. The navigation for the module is automatically generated based on the directory structure.

File Structure

[order-]title.html

Parameter Description
order
integer

* Optional

If provided, this will determine the page's order in the navigation

title
string

The title of the page, with words separated by hyphens.
